RedWhale – Bluetooth Chat Android Application

RedWhale is a simple Android application that allows two users to exchange text messages using Bluetooth.
This document explains the main features of the application and the technologies used to develop it.

Features
The application provides the following features:
Enable Bluetooth
Users can turn on the device’s Bluetooth directly from the app.
Search for Devices
The app can scan for nearby Bluetooth-enabled devices.
Display Device List
A list of paired and available Bluetooth devices is shown.
Connect to a Device
Users can select a device from the list and establish a connection.
Chat Function
After connecting, users can send and receive text messages through a simple chat interface.

Technologies Used
The RedWhale application is developed using the following technologies:
Programming Language: Java
Platform: Android
Build Tool: Gradle (Kotlin DSL / build.gradle.kts)
Supported Android Version: API level 24 and above (Android 7.0 / Nougat)
Android Core Components Used
The application uses the following Android components:
BluetoothAdapter
Used to manage Bluetooth operations such as enabling Bluetooth, discovering devices, and handling connections.
